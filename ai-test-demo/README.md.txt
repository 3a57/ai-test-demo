# AI Test Demo Project

## Day 1 Goal
- 搭建 Python 测试环境
- 使用 requests 调用公开 API

## 如何运行
1. `python -m venv venv`
2. `source venv/bin/activate` (Linux/Mac) 或 `venv\Scripts\activate` (Windows)
3. `pip install -r requirements.txt`
4. `python test_demo_api.py`
